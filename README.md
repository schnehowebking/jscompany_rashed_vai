## Installation

Install project with composer and yarn

```bash
  composer install
  ren .env.example .env
  yarn
  yarn dev
  yarn watch
```
    
## Note

- Add all backend controller inside Controllers/Backend folder
- Add all backend routes inside in backend.php
- Add all backend view files inside in views/backend
